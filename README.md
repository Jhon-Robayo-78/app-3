# Sistema de Control de Acceso Vehicular

Este es un proyecto creado con React utilizando la librería de bootstrap.

## ¿Qué es SCAV?

Con el uso del SCAV se ha mejorado la seguridad, integridad y organización 
del campus de la Universidad Tecnológica de Bolívar en cuanto al registro de 
vehículos. Con este sistema, y una base de datos con información detallada de 
cada vehículo que sale y entra al campus, además de esto, es posible encontrar 
estacionamientos libres y notificar al dueño del vehículo si es necesario.

### ¿Qué incorpora esta App?

Sistema de autenticación
Vista de usuario
Vista de Vigilante
Vista de controlador
